# elektrosak.com — Basit Flask elektronik mağaza demo

Yerel olarak Flask ile hazırlanmış küçük bir demo alışveriş sitesi.

Hızlı başlatma (Windows PowerShell):

```powershell
python -m pip install -r requirements.txt
python app.py
```

Uygulama çalışınca tarayıcıda http://127.0.0.1:5000 adresini açın.

Sayfalar:
- Anasayfa (ürün listesi örneği)
- Hakkımızda (`/hakkimizda`)
- İletişim (`/iletisim`) — form gönderince başarı sayfasına yönlendirir

Not: Bu proje demo amaçlıdır; üretim kullanımı için yapılandırma ve güvenlik ayarları ekleyin.
